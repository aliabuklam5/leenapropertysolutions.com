How it works: Each address is looked up directly in the Cape Coral GIS Address database (no third-party geocoder). The matched parcel point is then checked against Water district polygons, Sewer district polygons, the Zoning layer, the Environmental wildlife layer (Eagle/Owl/Tortoise buffers), and the Parcels layer for corner lot detection. Wildlife zones show Y if present, blank if not — matching what the IMS website displays.
